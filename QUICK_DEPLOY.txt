📦 RAILWAY (Backend) - 5 Steps
───────────────────────────────────────────────────────────────
1. Go to: https://railway.app
2. New Project → Deploy from GitHub → Select ELD-Trip-Planner
3. Settings → Root Directory → Type: backend
4. Variables → Add:
   DJANGO_SECRET_KEY = eld-trip-planner-secret-2024
   DEBUG = False
5. Copy your Railway URL (Settings → Domains)

───────────────────────────────────────────────────────────────

📦 VERCEL (Frontend) - 5 Steps
───────────────────────────────────────────────────────────────
1. Go to: https://vercel.com
2. Add New → Project → Import ELD-Trip-Planner
3. Root Directory → Edit → Type: frontend
4. Environment Variables → Add:
   REACT_APP_API_URL = [Your Railway URL from above]
5. Click Deploy → Wait 2 minutes → Done!
